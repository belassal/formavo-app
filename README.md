# Formavo App

Formavo is a youth sports team management platform built with React
Native and Firebase.

## Stack

-   React Native CLI
-   Firebase
-   Firestore (Realtime)
-   Firebase Auth
-   Native iOS Firebase SDK

## Current Version

v0.1 -- Firebase Foundation Build

## Features Implemented

-   Firebase native initialization (iOS)
-   Anonymous authentication
-   Secure Firestore rules
-   Realtime team listing
-   Modular service layer

## Setup Instructions

``` bash
npm install
cd ios
pod install
cd ..
npx react-native run-ios
```

## Firestore Structure

    clubs/{clubId}/teams/{teamId}
